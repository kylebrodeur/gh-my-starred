# Contributing to gh-my-starred

Thank you for your interest in contributing!

## Ways to Contribute

- Report bugs via [GitHub Issues](https://github.com/kylebrodeur/gh-my-starred/issues)
- Suggest features via [GitHub Issues](https://github.com/kylebrodeur/gh-my-starred/issues)
- Submit pull requests
- Improve documentation

## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/gh-my-starred.git
cd gh-my-starred

# Make executable
chmod +x gh-my-starred

# Test
bash test/run.sh
```

## Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Run tests (`bash test/run.sh`)
5. Commit with clear messages
6. Push to your fork
7. Open a Pull Request

## Code Style

- Bash: Follow POSIX where possible
- PI Extension: TypeScript with explicit types
- Keep it simple and focused
